## Default Search Order

When doing multi-instance searches, query in this order for best coverage:

1. `lemmy.ml` — flagship, broad federation
2. `lemmy.world` — largest user base
3. `programming.dev` — tech communities
4. `sh.itjust.works` — good general coverage
